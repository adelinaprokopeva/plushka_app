# Плюшка

## Описание проекта

Приложение "Плюшка" — это удобный инструмент для студентов и преподавателей ОмГТУ, позволяющий просматривать афишу мероприятий, управлять личным профилем, записываться на события и взаимодействовать с внутренними ресурсами учебного заведения.

Основные функции:
- Просмотр и регистрация на мероприятия
- Управление личным профилем с возможностью загрузки фотографии
- Регистрация и аутентификация пользователей с разными ролями (студент, преподаватель, администратор)

## Технологии

- Python 3
- PyQt6 для графического интерфейса
- SQLite для хранения данных
- Хеширование паролей с использованием SHA-256
- Markdown для документации

## Установка и запуск

1. Клонируйте репозиторий:
git clone <URL_репозитория>
2. Установите зависимости:
pip install -r requirements.txt
3. Запустите приложение
python main.py


## Структура проекта

- `ui/` — модули с графическим интерфейсом (окна входа, профиля, регистрации, мероприятий)
- `database.py` — работа с базой данных и логика хранения данных
- `models.py` — описание основных сущностей (пользователь, мероприятие)
- `main.py` — точка входа в приложение
